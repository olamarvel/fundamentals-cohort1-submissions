# DevConnect Frontend

This is the frontend for DevConnect, a platform for developers to connect, share projects, and collaborate.

## Features

*   User authentication (login and registration)
*   Create, edit, and view projects
*   A community page to see other users' projects
*   User profiles

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Node.js and npm installed. You can download them from [https://nodejs.org/](https://nodejs.org/).

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your_username/devconnect-frontend.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Start the development server
    ```sh
    npm run dev
    ```

The application will be available at [http://localhost:5173](http://localhost:5173).
